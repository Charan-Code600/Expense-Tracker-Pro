



# 💰 Expense Tracker Pro

A Python tool to track your daily income and expenses with CSV storage.

## Features
- ➕ Add Income
- ➖ Add Expense
- 💰 View Balance
- 📋 View History with Date
- 💾 Auto save & load from CSV
- 📅 Date tracking for every transaction

## How to Run
```bash
python expense_tracker.py
```

## Technologies Used
- Python
- Pandas
- DateTime
- CSV
